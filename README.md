⚽ Project World Cup Pong

A simple football-themed remake of the classic Pong game, developed in Unity for learning and study purposes.
The project mixes the nostalgic Pong gameplay with a World Cup visual style using pixel art aesthetics.

🎮 Features
⚽ Football-themed Pong gameplay
🕹️ Classic 2-player arcade mechanics
🎨 Pixel art inspired visuals
🏟️ Soccer field arena design
🔊 Sound effects and score system
📋 Main Menu and Game Over screens
✨ Simple and clean UI/HUD
🛠️ Technologies Used
Unity Engine
C#
TextMeshPro
DOTween (UI animations and juice effects)
📷 Preview

Game based on the famous classic PONG, redesigned with a football/World Cup atmosphere.
